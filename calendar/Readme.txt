#Calendar application using fullcalendar plugin
-----------------------------------------------------------------------------------------------------------------------

Reference: https://fullcalendar.io/docs

-----------------------------------------------------------------------------------------------------------------------

#steps to use plugin

-Download the zip file of the plugin.
-unzip it in a folder
-Place the cal.html file inside the lib folder in it so that the style sheets and javascripts files can be accessed.

-----------------------------------------------------------------------------------------------------------------------

#About  the application

-This application opens up a calender in week mode with time grids.
-It has some default events which can be dragged and dropped on some day of a week.
-One can resize the event such that it covers a time duration.
